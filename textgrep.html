Tool Arsenal

Core Search

  • grep — Pattern search with line numbers
  • grep_context — Include N lines before/after matches
  • grep_files — Return only filenames containing pattern
  • grep_count — Count occurrences per file

Extraction

  • extract_matches — Pull regex capture groups, optional unique filter
  • extract_lines — Get specific line ranges
  • filter_lines — Keep/remove lines matching pattern

Analysis

  • file_stats — Lines, words, chars, encoding detection
  • compare_files — Unified diff between two files
  • find_duplicates — Locate duplicate lines across files

Modification

  • replace — Preview regex replacements before executing
  • replace_confirm — Execute previewed replacement
